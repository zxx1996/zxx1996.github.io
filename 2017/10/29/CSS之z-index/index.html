z-index属性

控制标签的层级关系，最底层的数值最小，最顶层的数值最大.通常当定义的CSS中有postion属性为absolute，relative或fixed时，用z-index会生效。数值越大时被层叠在最上边

z1,z2,z3是同等大小的div层，设置z-index属性不同，此时z3的z-index值比其它层大，页面显示z3层的内容
