前端

  • HTML :描述网页语言;超文本标记语言; 不是编程语言，是一种标记语言，与C++不相通

HTML5

HTML5的优点:

  1. 跨平台 PC 手机 Linux
  2. 硬件要求低
  3. flash之外的选择 尤其是手机端
  4. HTML5 ：用于绘画canvas 标签；媒体 video audio元素；本地离线储存；

HTML基础

  • 新建项目：Create New Project–Static web–HelloWord

    New-->HTML File-->index

  • 设置快捷键： Preferences(Setting)–Keymap 快捷键列表

    (代码补全 ：Alt+/)   (代码上移或下移：Alt+上下)
(代码快速复制：Alt+Ctrl+上下)   (删除一行：Ctrl+D)

  • js文件 引用:

  • 声明：<!DOCTYPE>

    HTML5:  <!DOCTYPE html>     声明代表是HTML5版本

  • 元素：从开始标签到结束标签的所有代码

  • 头标记：

     <head></head>
编码格式： <meta charset="UTF-8">
标题：<title>网页标题</title>
引用CSS样式：<style type=" "></style>

  • body：

    <body></body>------bgcolor 背景颜色      background 背景图片
HTML标题: '<h1>'...'<h6>'  --------align 对齐方式(默认居左)
段落标记(另起)：<p></p>
超链接：<a href="http://www.baidu.com">百度</a>-----target 在何处打开链接(覆盖)
图像标记：<img src="images/html.png"></img>
空标签：<br/>  换行
  • 通用标签属性：
    id :规定元素唯一ID  (后期引用要使用)    class:规定元素类名         i
style:规定元素样式(引用外部文件)         title:规定元素额外信息
