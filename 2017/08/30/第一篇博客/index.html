搭建博客网站

（这可能是在只有我自己才能看懂的笔记(●’◡’●)）

安装 Node 和 Git

注册 GitHub 账号

  • 获取 SSH Keys，配置到 config.yml 文件中。

新建网站文件夹

运行命令（在桌面右击鼠标Git Bash Here）

  • node -v (检测node是否成功）
  • git version （检测git版本）
  • npm install -g hexo （安装hexo）
  • hexo -v （hexo组件检测）

文件夹右击 执行命令

hexo init （克隆hexo到⾃⼰博客⽂件夹）
hexo s （开启本地调试模式）
npm install hexo-deployer-git –save
ssh-keygen -t rsa -C “986211552@qq.com” 生成密钥 ，复制到GitHub中

git config –global user.name “dellcorsair” （换成⾃⼰GitHub账户）
git config –global user.email “986211552@qq.com” （自己邮箱）
ssh -T git@github.com （和GitHub存储仓库的链接测试）
git config –list （再次查看配置是否正确）
最后发布：hexo clean && hexo g -d
