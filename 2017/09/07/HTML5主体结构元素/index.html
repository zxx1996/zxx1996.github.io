HTML5语义化标签

发表于 2017-09-07 | 分类于 HTML

标签

  • aricle元素

    文档，页面或应用程序中独立的、完整的、可以独自被外部引用的内容。可嵌套，可代替DIV，内嵌页面，可以看成特殊的section元素，比section元素更加强调独立性

  • section元素

    用于对网站或应用程序中页面上的内容进行分块。对文章进行分段，强调分段、分行。
    1.不要将section元素作为设置页面样式的容器。
    2.若article元素、aside元素或nav元素更符合使用条件，不要使用section元素。
    3.不要为没有标题的内容区块使用section元素。

  • nav元素

    可用作页面导航的链接组，其中导航元素链接到其他页面或当前页面的其他部分。
    一个页面可以有多个nav元素。
    注：nav元素是用在一系列发出命令的菜单上的，是一种交互性的元素，使用在WEB应用程序中

    • 传统导航条
    • 侧边导航
    • 页内导航
    • 翻页操作

  • 页面：

  • aside元素：

    用来表示当前页面或文章的附属信息部分，他可以包含与当前页面或主要内容的相关引用、侧边栏、广告、导航条，以及其他类似的有区别于主要内容的部分。
