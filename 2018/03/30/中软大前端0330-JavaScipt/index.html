    typeOf()：获取变量的数据类型，以字符串形式返回。只能识别5种基础数据类型和对象类型（object）

    数据类型

    基础数据类型

    • undefined类型
      未定义的类型，只声明，未赋值。（缺少值）

    • null类型
      null数值类型，在与number类型数据做运算时，null转换成数值0

    • boolean类型
      其他数据类型转换成boolean类型
      var boo=Boolean(其他数据类型)
      String =””; ——false
      number = 0; ——false
      非null ——true
      null ——false

    • number类型
      16位数
      转换-内置函数
      parseInt(“字符”,进制数);
      paserFloat(“字符”,进制数)
      默认为十进制
